# Examples

Runnable examples demonstrating `hiero-sdk-utils` against the live Hedera testnet.

## Prerequisites

Run `npm install` from the repo root first:

```bash
cd ..  # repo root
npm install
```

Then run any example with:

```bash
npx tsx basic-queries.ts
npx tsx pagination.ts
npx tsx error-handling.ts
```

No credentials required. All examples use the public Hedera testnet Mirror Node.
